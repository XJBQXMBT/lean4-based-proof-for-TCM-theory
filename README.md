#《基于范畴论探讨中医疫病共性病机数学内涵》命题1至4的集中证明

#需要依赖Mathlib定理库(版本最晚日期20260201)

#（1）临床命题“同一疾病，在单一辨证主体或辨证体系下有且仅有唯一的核心病机”转化为数学命题1；

#（2）临床命题“同一疾病的不同症状表现、辨证分型，均可以由核心病机解释”“同一疾病的不同症状表现、辨证分型，其由核心病机的解释不同”转化为数学命题2；

#（3）临床命题“多个疾病存在共性病机，则它们首先存在相似的疾病阶段”“多个疾病存在共性病机，则在每个相似疾病阶段的症状是相似的”转化为数学命题3；

#（4）临床命题“多个疾病存在共性病机，则在每个相似疾病阶段内具有唯一的共同病机”“共性病机是多个整体病机的共性成分，其也应是整体的形式”转化为数学命题4。
